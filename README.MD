逆向迅雷影音字幕API，封装为单独字幕下载器。

本REPO下有C#版与Electron版，但二者均为当初练手所写，代码质量很糟糕，实现也较为简单，有兴趣的朋友可以自行实现。

已经很久没有使用Windows，故将不会继续维护这两个版本，目前主要维护的将会是[Rust命令行版](https://github.com/DCjanus/lone-ranger)。
